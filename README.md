# Collaborative Drawing Canvas

A real-time multi-user drawing application built with JavaScript, Node.js, and Socket.io.

## Features

- **Real-time Drawing**: Multiple users can draw simultaneously on the same canvas
- **Multiple Tools**: Brush and eraser with adjustable size and color
- **User Management**: See who's online with colored indicators
- **Room System**: Isolated drawing sessions with room IDs
- **Undo/Redo**: Global undo/redo functionality across all users
- **Mobile Support**: Touch events for drawing on mobile devices
